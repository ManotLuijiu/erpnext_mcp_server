About MCP Server

                

                

                    
The Model Context Protocol (MCP) server allows AI models to interact with ERPNext through a standardized interface. It enables your chat bot to perform operations such as:

                    
    
                        
  • Querying ERPNext documents (like Sales Invoices, Customers, etc.)
    • 
                        
  • Creating and updating documents
    • 
                        
  • Running reports and analyses
    • 
                        
  • Executing workflows and custom functions
    • 
                    

                    
Start the server to enable AI integration with your ERPNext system.
